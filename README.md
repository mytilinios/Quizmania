# QuizMania 🎮

Multiplayer Quiz Game με AI ερωτήσεις, μουσική και σύστημα πόντων ταχύτητας.

## Οδηγίες Ανεβάσματος

### Βήμα 1: Δημιουργία αποθετηρίου στο GitHub
1. Πήγαινε στο github.com και κάνε login
2. Πάτα το "+" πάνω δεξιά → "New repository"
3. Όνομα: `quizmania`
4. Επίλεξε "Public"
5. Πάτα "Create repository"

### Βήμα 2: Ανέβασμα αρχείων
1. Στη σελίδα του repository, πάτα "uploading an existing file"
2. Σύρε ΟΛΑ τα αρχεία (server.js, package.json, railway.toml, και τον φάκελο public/)
3. Πάτα "Commit changes"

### Βήμα 3: Deploy στο Railway
1. Πήγαινε στο railway.app
2. Πάτα "New Project"
3. Επίλεξε "Deploy from GitHub repo"
4. Επίλεξε το repository "quizmania"
5. Το Railway θα ξεκινήσει αυτόματα!

### Βήμα 4: Το domain σου
1. Στο Railway project, πάτα "Settings"
2. Πάτα "Generate Domain"
3. Πάρεις το link σου (π.χ. quizmania-production.up.railway.app)

## Πώς παίζεται
- **Host**: Ανοίγει το link, πατάει "Δημιουργία Δωματίου"
- **Παίκτες**: Ανοίγουν το ίδιο link από το κινητό τους, πατάνε "Είσοδος" και βάζουν τον κωδικό
- Ο Host επιλέγει κατηγορία και ξεκινάει το παιχνίδι
